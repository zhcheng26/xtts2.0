# 声音克隆 + 歌声生成系统设计文档

**日期**：2026-04-07  
**项目**：xtts2.0  
**方案**：RVC v2 Pipeline + Gradio UI

---

## 1. 需求概述

构建一个完整的声音克隆 + 翻唱生成系统：
- 用户上传目标人的声音样本（10~30分钟干净人声），微调 RVC v2 模型
- 用户上传任意中文歌曲（MP3/WAV），系统自动分离人声与伴奏
- 将分离出的人声转换为目标人声音色，再与伴奏混合输出最终翻唱音频
- 通过 Gradio Web UI 提供完整交互体验

**运行环境**：RTX 2080 Ti，CUDA 11.8，VRAM 24GB，本地运行

---

## 2. 技术选型

| 组件 | 工具 | 版本/说明 |
|------|------|----------|
| 人声分离 | Demucs | htdemucs_ft 模型，SOTA 分离质量 |
| 音高提取 | RMVPE | 比 crepe/harvest 更准确的 F0 提取 |
| 声音转换 | RVC v2 | 最成熟的歌声转换方案，社区活跃 |
| 音频混合 | pydub + ffmpeg | 人声与伴奏混合，支持音量调节 |
| UI 框架 | Gradio | 本地 Web Demo |
| 深度学习框架 | PyTorch + CUDA 11.8 | GPU 加速 |

---

## 3. 整体架构

系统分为两个独立阶段：训练阶段和推理阶段。训练一次模型，可反复用于多首歌曲翻唱。

```
┌─────────────────────────────────────────────────────┐
│                   Gradio Web UI                      │
│  [声音样本上传] [歌曲上传] [训练] [推理] [结果播放]  │
└────────────────────┬────────────────────────────────┘
                     │
        ┌────────────┴────────────┐
        │                        │
   ┌────▼─────┐           ┌──────▼──────┐
   │ 训练模块  │           │   推理模块   │
   │          │           │             │
   │ 1.音频预处理│          │ 1.人声分离   │
   │ 2.特征提取 │           │  (Demucs)  │
   │ 3.RVC微调 │           │ 2.音高提取  │
   └──────────┘           │  (RMVPE)   │
                          │ 3.RVC推理   │
                          │ 4.音频混合  │
                          └─────────────┘
```

---

## 4. 数据流

### 训练数据流
```
voice_samples/ (10~30分钟干净人声音频)
  → 音频切片（每段3~10秒）+ 降噪预处理
  → HuBERT 特征提取
  → RVC v2 fine-tune（约30~60分钟，on RTX 2080 Ti）
  → model.pth（模型权重）+ index.bin（特征索引）
```

### 推理数据流
```
song.mp3
  → Demucs (htdemucs_ft) → vocals.wav + accompaniment.wav
  → RMVPE → f0.npy（F0 音高曲线）
  → RVC v2 推理（使用已训练模型）→ converted_vocals.wav
  → pydub 混合（音量可调）→ final_output.wav
```

## 6. Gradio UI 设计

### Tab 1 - 模型训练
- **模型名称**：文本输入，用于标识和保存模型
- **声音样本**：多文件上传（MP3/WAV），10~30分钟干净人声
- **训练轮数**：滑块，默认 200 轮
- **批大小**：数字输入，默认 4
- **开始训练**：按钮，触发异步训练
- **训练日志**：实时滚动文本框，显示训练进度

### Tab 2 - 歌声生成
- **选择模型**：下拉框，列出 `models/` 目录下已训练模型
- **上传歌曲**：单文件上传（MP3/WAV）
- **音调偏移**：滑块，-12 ~ +12 半音，默认 0（用于调整男女音域差异）
- **人声音量**：滑块，0.0 ~ 2.0，默认 1.0
- **伴奏音量**：滑块，0.0 ~ 2.0，默认 1.0
- **开始生成**：按钮，触发推理流程
- **结果展示**：三个音频播放器（原始人声 / 转换后人声 / 最终混音）+ 下载按钮

---

## 7. 关键参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 训练轮数 | 200 | 样本少时可降至 100，多时可升至 400 |
| 批大小 | 4 | 2080 Ti 下稳定不 OOM |
| 音调偏移 | 0 | 女翻男通常 -5~-12，男翻女通常 +5~+12 |
| index_rate | 0.75 | RVC 特征索引混合比例，影响音色相似度 |
| f0_method | rmvpe | 音高提取方法，RMVPE 精度最高 |

---

## 8. 约束与边界

- 仅支持中文歌曲（系统设计未针对多语言优化）
- 声音样本需为干净人声（无背景音乐），否则训练效果下降
- 训练约需 30~60 分钟，推理约需 1~3 分钟（视歌曲时长）
- 不支持实时推理，为批处理模式
